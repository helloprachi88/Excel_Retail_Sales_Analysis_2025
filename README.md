# Retail Sales Analysis Report 2025

## Project Overview
This project analyzes retail sales data for 2025 using Microsoft Excel.  
The dashboard provides insights into sales performance, order trends, customer gender split, product sizes, sales channels, categories, and top-performing states.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Formatting

## Dashboard Features
- Sales comparison between Men and Women
- Monthly Order vs Sales trend
- Size-wise quantity sold
- Channel-wise sales amount
- Top 5 states by sales
- Order sales channel distribution
- Interactive slicers for:
  - Category
  - Channel
  - Month

## Key Insights
- Women contributed the highest sales share at 64%.
- Amazon was the top sales channel with around 35% of total orders.
- Maharashtra and Karnataka were among the top-performing states.
- Myntra and Flipkart also contributed significantly to overall sales.
- Popular sizes included M, L, XL, and XXL.
- Ethnic Dress, Kurta, Saree, Set, Top, and Western Dress were major product categories.
